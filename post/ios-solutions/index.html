一些问题的解决方案

创建: 2021-07-16   |   更新: 2021-07-16   |   分类: iOS     |   字数: 1557   |   阅读: 4 分钟

在开发过程中，经常会遇到各种问题，有时候某个问题虽然已经找到了解决方案，但过了段时间又会忘记，这篇文章主要记录一些个人认为需要记录的问题的解决方案。

强制竖屏

最近在调其它SDK的时候，要求某个界面是竖屏的，发现按约定传了相关的参数，还是会横竖屏切换。于是告诉对方，首先在工程的info.plist中添加Require Full Screen，并将值置为Yes，然后在ViewController中添加如下代码：

class ViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        let value = UIInterfaceOrientation.portrait.rawValue
        UIDevice.current.setValue(value, forKey: "orientation")
    }
    
    override var supportedInterfaceOrientations: UIInterfaceOrientationMask {
        return .portrait
    }

    override var shouldAutorotate: Bool {
        return true
    }
}

keyWindow变化

升级到iOS 14.5.1后发现使用UIApplication.shared.keyWindow获取RootViewController为nil。经排查发现keyWindow发生了变化，为了防止这种变化，在显示新的Window时，先保存keyWindow，释放新后再还原即可：

func show() {
    if self.showTimes == 0 {
        // iOS 14.5.1上如果不记录keyWindow，调用self.dismissVC()后会导致dismiss后keyWindow发生变化
        // 从而导致取topMost时为nil，路由不能正常push或present
        self.keyWindow = UIApplication.shared.keyWindow
        let window = UIWindow(frame: UIScreen.main.bounds)
        window.windowLevel = UIWindow.Level.normal
        window.safeRootViewController = self
        window.isHidden = false
        window.backgroundColor = UIColor.clear
        self.newWindow = window
        self.showTimes += 1
    }
}

@IBAction func cancelAction(_ sender: Any) {
    self.dismissVC()
    // 恢复keyWindow，以解决iOS 14.5.1 keyWindow发生变化的情况
    self.keyWindow?.makeKeyAndVisible()
}

禁用返回手势

对于push的界面，默认有「边缘返回」手势。但有时候需要禁止系统的「边缘返回」手势，可以为UIViewController写一个扩展方法，然后按需使用：

// 参考自https://juejin.cn/post/6860656306630590477
extension UIViewController {
    /// 禁止系统返回手势
    func popGestureForbid() {
        if let gestures = self.navigationController?.interactivePopGestureRecognizer?.view?.gestureRecognizers {
            gestures.forEach({ $0.isEnabled = false })
        }
    }
}

webView特殊字符

如果UIWebview中加载的html文件中含有\n,'等特殊字符时，将会看到为一片空白。因此加载之前，需要将这些特殊字符进行如下处理:

class HTMLUtils {
    static func escapsedHTML(html: String) -> String{
        var result = html
        if !result.isEmpty{
            result = result.replacingOccurrences(of: "\\", with: "\\\\")
            result = result.replacingOccurrences(of: "\"", with: "\\\"")
            result = result.replacingOccurrences(of: "'", with: "\\'")
            result = result.replacingOccurrences(of: "\r", with: "\\r")
            result = result.replacingOccurrences(of: "\n", with: "\\n")
        }
        return result
    }
}
