基础

文件扩展名

所有的Ruby文件扩展名都是.rb。在ruby代码中，空白字符一般都会被忽略，而字符串中的空白字符不会被忽略。

变量

在Ruby中，变量不需要提前声明，为变量赋值时会自动创建。例如，a = 300 会创建一个名为a的变量，并将300存储在该变量中。

Ruby变量可以容纳任何类型的数据，甚至可以对一个已经定义的变量，更改其持有的数据类型，如下所示：

test = 45
puts test
test = "test"
puts test

Ruby中变量名的风格偏好：使用小写字母，以下划线连接，不使用缩写，不以数字开头，以内容的含义命名。

字符串

Ruby中字符串分为单引号和双引号字符串，区别在于双引号字符串能够支持更多的转义字符。

单引号

如果需要在单引号字符串中使用单引号，必须使用转义字符，例如'Won\'t you read O\'Reilly\'s book?'
